# 📊 Student Data Analysis Project

## 📌 Overview
This project analyzes student performance data using Python.  
It includes data loading, cleaning, transformation, analysis, and reporting.

---

## ⚙️ Features
- Load dataset (1000 student records)
- Handle missing values
- Feature engineering (new insights)
- Data visualization and analysis
- Generate final report

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Analysis Performed
- Study Hours vs Marks relationship
- Attendance impact on performance
- Average marks calculation
- Top performing students identification

## 🚀 How to Run

1. Clone the repository:
```bash
git clone <repository-link>
```

2. Go to the project folder:
```bash
cd Student-Data-Analysis-Project
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the project:
```bash
python main.py
```

## 📊 Expected Output

### 📈 Key Insights
- Students with higher study hours perform better.
- Attendance strongly impacts marks.
- Top students have consistent study habits and attendance.

### 📋 Results Summary
- Average performance varies around the mid-high range.
- Study hours are the strongest factor.
- Attendance also plays an important role.

This project helps analyze student performance patterns using data science techniques.
